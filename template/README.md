# {{PROJECT_NAME}}

A Spring Boot application based on the monorepo template.

## Prerequisites

- Java 21 or higher
- Gradle (wrapper included)

## Building the Project

Build the project using Gradle:

```bash
./gradlew build
```

## Running the Application

Run the application locally:

```bash
./gradlew bootRun
```

## Testing

Run all tests:

```bash
./gradlew test
```

## Cleaning Build Artifacts

Clean the build directory:

```bash
./gradlew clean
```
